This small script will allow you to download list of youtube videos using youtube_dl python library.

Usage
1. First install the youtube_dl library
    pip install youtube_dl

2. Then you can need to insert the list of youtube urls in to url_list.csv file. Each url needs to be in a new line

3. Run the script to start the download
    python downloader.py

4. Videos in the url list will be downloaded in to the root where this script resides in your local machine.
